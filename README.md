# Student Management System  
A Java-based Student Management System with CRUD operations using MySQL.  

## Features  
- Add Student  
- View Students  
- Update Student  
- Delete Student  

## Technologies Used  
- Java  
- MySQL  
- JDBC  

## Setup Instructions  
1. Clone the repository:  
   ```sh
   git clone https://github.com/2300040100/Student-Management-System.git
   ```
2. Open in Eclipse  
3. Configure MySQL and run the application
